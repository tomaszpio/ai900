Natural Language Processing

Assess your understanding of Azure AI services that analyze and generate language.

1️⃣ Which Azure AI service provides prebuilt capabilities for sentiment analysis and entity recognition?

  • A. Azure AI Language
  • B. Azure Cognitive Search
  • C. Azure Form Recognizer
  • D. Azure Synapse Analytics
Show Answer
✅ Correct Answer: A — Azure AI Language.
Explanation: Azure AI Language offers prebuilt and customizable models for sentiment, entities, key phrases, and more.

2️⃣ What is a primary benefit of using Azure OpenAI Service for conversational bots?

  • A. Automatic indexing of PDFs
  • B. Large language models that can follow instructions and maintain context
  • C. Built-in speech recognition
  • D. Visual anomaly detection
Show Answer
✅ Correct Answer: B — Large language models that can follow instructions and maintain context.
Explanation: Azure OpenAI provides advanced generative models ideal for conversational agents that need nuanced understanding and responses.

3️⃣ Which feature of Language Studio lets you build a custom text classification model without writing code?

  • A. Custom Named Entity Recognition
  • B. Conversational Language Understanding
  • C. Question Answering
  • D. Orchestration workflow
Show Answer
✅ Correct Answer: B — Conversational Language Understanding.
Explanation: Conversational Language Understanding allows you to label intents and entities via a visual interface, creating custom models for natural language input.

4️⃣ When integrating Azure Cognitive Search with language models, what role does the search index play?

  • A. It generates responses directly.
  • B. It provides grounding documents that the model can reference.
  • C. It trains the model weights.
  • D. It converts audio to text.
Show Answer
✅ Correct Answer: B — It provides grounding documents that the model can reference.
Explanation: Azure Cognitive Search retrieves relevant content, which can then be passed to a language model to ground generated answers in trusted data.

5️⃣ Which capability ensures that a multilingual chatbot built on Azure can detect the language of incoming messages automatically?

  • A. Speech translation
  • B. Language detection
  • C. Translator custom glossaries
  • D. Bot Framework SDK
Show Answer
✅ Correct Answer: B — Language detection.
Explanation: Azure AI Language offers language detection so solutions can route or translate messages appropriately without manual selection.
