Responsible AI

Review the principles, tools, and governance practices that ensure AI solutions remain trustworthy.

1️⃣ Which Microsoft Responsible AI principle focuses on protecting personal data and privacy?

  • A. Fairness
  • B. Reliability & Safety
  • C. Privacy & Security
  • D. Inclusiveness
Show Answer
✅ Correct Answer: C — Privacy & Security.
Explanation: The Privacy & Security principle emphasizes safeguarding data and ensuring AI systems protect sensitive information. Learn more in the Microsoft documentation.

2️⃣ What is the purpose of an impact assessment in Responsible AI governance?

  • A. To measure model accuracy
  • B. To document potential harms, mitigations, and accountability
  • C. To select cloud regions for deployment
  • D. To configure monitoring alerts
Show Answer
✅ Correct Answer: B — To document potential harms, mitigations, and accountability.
Explanation: Impact assessments capture the societal and ethical considerations of an AI system, guiding responsible deployment decisions. Learn more in the Microsoft documentation.

3️⃣ Which Azure service helps detect and mitigate bias in machine learning models?

  • A. Azure Machine Learning responsible AI dashboard
  • B. Azure Monitor
  • C. Azure Key Vault
  • D. Azure Policy
Show Answer
✅ Correct Answer: A — Azure Machine Learning responsible AI dashboard.
Explanation: The responsible AI dashboard provides fairness, interpretability, and error analysis tools to identify and reduce bias in models. Learn more in the Microsoft documentation.

4️⃣ Which practice ensures users understand how AI-driven decisions are made?

  • A. Using black-box models
  • B. Providing transparency and interpretability reports
  • C. Encrypting all data
  • D. Deploying to multiple regions
Show Answer
✅ Correct Answer: B — Providing transparency and interpretability reports.
Explanation: Transparency requires sharing explanations, documentation, and communication so stakeholders understand model outputs and limitations. Learn more in the Microsoft documentation.

5️⃣ How can you maintain Responsible AI standards after deploying a model?

  • A. Disable logging to protect privacy
  • B. Establish continuous monitoring and human oversight
  • C. Remove the impact assessment
  • D. Only allow autonomous retraining
Show Answer
✅ Correct Answer: B — Establish continuous monitoring and human oversight.
Explanation: Ongoing monitoring with human-in-the-loop review catches drifts, anomalies, and policy violations, keeping the solution aligned with Responsible AI commitments. Learn more in the Microsoft documentation.
