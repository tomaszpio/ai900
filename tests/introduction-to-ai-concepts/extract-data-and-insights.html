Extract Data and Insights

Test your knowledge of Azure services that help discover information from documents and data stores.

1️⃣ Which service can analyze scanned receipts to extract totals, dates, and merchant names?

  • A. Azure Cognitive Search
  • B. Azure Form Recognizer
  • C. Azure Machine Learning
  • D. Azure Databricks
Show Answer
✅ Correct Answer: B — Azure Form Recognizer.
Explanation: Form Recognizer provides prebuilt models for receipts, invoices, and other business documents, extracting key-value pairs with high accuracy. Learn more: Microsoft Learn — Azure AI Document Intelligence overview.

2️⃣ Why would you enrich documents in an Azure Cognitive Search index?

  • A. To increase storage redundancy
  • B. To add AI-generated metadata such as key phrases and entities
  • C. To convert images to SVG format
  • D. To replicate the index across regions
Show Answer
✅ Correct Answer: B — To add AI-generated metadata such as key phrases and entities.
Explanation: Skillsets enrich content during indexing, creating searchable fields that surface insights from unstructured data. Learn more: Microsoft Learn — Cognitive search enrichment.

3️⃣ Which component of Azure Cognitive Search defines the schema for searchable content?

  • A. Indexer
  • B. Skillset
  • C. Index
  • D. Data source
Show Answer
✅ Correct Answer: C — Index.
Explanation: The index describes fields, data types, and search behavior, acting as the schema that clients query. Learn more: Microsoft Learn — Azure Cognitive Search index concepts.

4️⃣ You need to build a pipeline that ingests PDFs from Azure Blob Storage and makes them searchable. What should you configure first?

  • A. A data source pointing to the storage account
  • B. A semantic ranker
  • C. A private endpoint
  • D. Azure Synapse Link
Show Answer
✅ Correct Answer: A — A data source pointing to the storage account.
Explanation: The data source connection tells Azure Cognitive Search where to retrieve content before indexers and skillsets process the files. Learn more: Microsoft Learn — Azure Cognitive Search data sources.

5️⃣ What advantage does Azure Synapse Analytics provide when analyzing insights extracted from AI workloads?

  • A. Built-in labeling tools for training models
  • B. Serverless SQL and Spark pools for large-scale analytics
  • C. Automatic speech translation
  • D. Managed conversational bots
Show Answer
✅ Correct Answer: B — Serverless SQL and Spark pools for large-scale analytics.
Explanation: Synapse unifies big data and data warehousing, enabling teams to analyze extracted insights alongside enterprise data. Learn more: Microsoft Learn — Azure Synapse Analytics overview.
