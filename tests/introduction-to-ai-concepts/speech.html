Speech Workloads

This quiz focuses on Azure AI services that convert, translate, and analyze speech.

1️⃣ Which Azure service provides speech-to-text transcription with custom vocabularies?

  • A. Azure AI Language
  • B. Azure Speech Service
  • C. Azure OpenAI Service
  • D. Azure Bot Service
Show Answer
✅ Correct Answer: B — Azure Speech Service.
Explanation: The Speech service supports real-time and batch transcription and allows custom speech models for unique vocabularies. Learn more in the Microsoft documentation.

2️⃣ What feature of Azure Speech enables real-time translation of spoken audio into multiple languages?

  • A. Speech studio conversation transcription
  • B. Speech translation
  • C. Neural Text-to-Speech
  • D. Custom Neural Voice
Show Answer
✅ Correct Answer: B — Speech translation.
Explanation: Speech translation can translate incoming speech into text or synthetic speech in target languages in near real-time. Learn more in the Microsoft documentation.

3️⃣ Which capability lets you build a branded synthetic voice using your own recordings?

  • A. Custom Neural Voice
  • B. Keyword recognition
  • C. Audio content safety
  • D. Speech Synthesis Markup Language
Show Answer
✅ Correct Answer: A — Custom Neural Voice.
Explanation: Custom Neural Voice lets organizations create high-quality, realistic synthetic voices trained on their own audio datasets. Learn more in the Microsoft documentation.

4️⃣ When should you use the batch transcription API instead of real-time speech-to-text?

  • A. For conversations that need speaker diarization
  • B. For streaming voice commands in an app
  • C. For low-latency transcription in a call center
  • D. For summarizing recorded meetings stored in Azure Blob Storage
Show Answer
✅ Correct Answer: D — For summarizing recorded meetings stored in Azure Blob Storage.
Explanation: Batch transcription handles large prerecorded files asynchronously, ideal for archived meeting recordings. Learn more in the Microsoft documentation.

5️⃣ Which security feature helps prevent unauthorized use of a custom voice model?

  • A. Content filters
  • B. Resource locks
  • C. Speaker verification
  • D. Voice talent agreements
Show Answer
✅ Correct Answer: D — Voice talent agreements.
Explanation: Custom Neural Voice requires documented consent from the voice talent, ensuring ethical usage and preventing misuse of the voice model. Learn more in the Microsoft documentation.
